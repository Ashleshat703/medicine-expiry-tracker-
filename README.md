# Momentum Studio

A lightweight starter project built with plain HTML, CSS, and JavaScript.

## What is included

- A responsive landing page
- A small interaction layer for theme switching
- A note area with a live character counter
- A visual style that is easy to customize

## Run it

Open `index.html` in a browser.

If you want a local server from PowerShell, you can run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Project structure

- `index.html` contains the page structure
- `styles.css` contains the visual system and responsive layout
- `script.js` contains the interactive behavior
