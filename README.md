## Features

- Medicine inventory dataset in CSV format
- SQL script for database creation and analysis
- Power BI dashboard files for visualization
- Expiry date tracking
- Stock analysis by place, category, and supplier
- Total inventory value calculation

## Tech Stack

- SQL
- MySQL
- Power BI
- CSV Dataset

### SQL Analysis Included
The SQL file includes queries for:

Creating the medicine inventory database
Creating the medicine_inventory table
Counting total records
Viewing sample rows
Finding expired medicines
Finding medicines expiring within 180 days
Stock analysis by place
Stock analysis by category
Stock analysis by supplier
Calculating total inventory value


### Power BI Dashboard Use
The Power BI files can be used to:

Build stock monitoring dashboards
Visualize inventory distribution
Track expiry trends
Analyze supplier and category performance
Support inventory decision-making


### Getting Started
Clone or download this repository
Import the CSV file into your SQL database or Power BI
Run the SQL queries from sql queries/medicine.sql
Open the .pbix files in Power BI Desktop to explore the dashboards
